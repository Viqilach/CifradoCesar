# CifradoCésar

## Descripción

Este proyecto consiste en un programa en Java que permite cifrar y descifrar un mensaje utilizando el algoritmo de **Cifrado César**.

El usuario introduce un texto y una clave numérica (desplazamiento).  
Cada letra del mensaje se mueve un número de posiciones en el alfabeto según la clave indicada.

Si el desplazamiento supera la letra **Z**, el programa vuelve a empezar desde **A**.

Los **espacios y caracteres especiales** se mantienen sin modificar.

## Tecnologías utilizadas

- Java
- Scanner (entrada por teclado)
- StringBuilder (para construir el texto cifrado de forma más eficiente)
